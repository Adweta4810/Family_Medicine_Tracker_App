# Family Medicine Reminder App

## Overview
Family Medicine Reminder App is an Android application designed to help families manage and track medications for multiple members in one place. The app allows users to set reminders, store prescription details, and monitor taken or missed doses efficiently. It is especially useful for elderly care and families who need structured medicine management.

---

## Features

### Home Dashboard
- View upcoming reminders (e.g., Grandmother – 1 min left (Medicine Name))
- Quick add reminder option
- Mark medicine as Taken or Snooze
- Daily summary: Taken / Pending / Missed

### Login & Security
- Sign up and login using Email + Password
- Social login: Google (Android) and Apple
- OTP / Email verification
- Family account system (one account manages multiple members)

### Medicine Details Management
- Add medicines for different family members
- Store medicine details: Name, Dosage (mg), Frequency, Duration
- Notes section (Before/After food, special instructions)
- Upload prescription and medicine images
- Store doctor and hospital information

### Smart Reminder System
- Set reminder by Person + Medicine + Time
- Notification with sound and vibration
- Snooze option (5 / 10 minutes)
- Optional caregiver notification if dose is not marked as taken

### Medicine Tracker
- Track Taken, Missed, and Changed medicines
- Calendar-based history view
- Monthly adherence report (e.g., 90% taken)
- Optional medicine stock alert (low stock reminder)

---

## Objective
To simplify family medication management by providing timely reminders, organized medicine records, and adherence tracking in a user-friendly Android application.
